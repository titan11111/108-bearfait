# PIXEL QUEST — 世界観・アート・音楽 指示目次

画像やBGMを発注・生成する際のベースとして参照する目次です。各見出しごとに「何を依頼するか」のチェックと、指示文のたたき台を書けます。

---

## 1. ゲーム全体の雰囲気・世界観

| 項目 | 内容 |
|------|------|
| **タイトル** | PIXEL QUEST |
| **ジャンル** | 2D横スクロールアクション（コイン収集・敵撃破・ステージボス・ゴール到達） |
| **基調** | レトロファンタジー＋ピクセルアート。夜〜夕暮れの「冒険の旅」が少しずつ闘いの色を増していくイメージ。 |
| **トーン** | 懐かしさのある16bit風。過度にコミカルではなく、少しシリアスな剣と魔法の世界。 |
| **解像度・形式** | ゲーム内は 640×400、タイル16px。画像発注時は「16px単位のピクセルアート」「image-rendering: pixelated を想定」と伝えるとよい。 |

**指示文の例（画像用）**  
「16bit風ピクセルアート、横スクロールアクションゲーム用。レトロファンタジーで、夜の空とシルエットの山・木が背景。やや暗めの青〜紫ベースで、アクセントは金（#ffcc00）と赤。」

**指示文の例（音楽用）**  
「レトロRPG風の2Dアクションゲーム。冒険の始まりは穏やか、中盤で戦闘・決意、ラストはボス戦のクライマックス。8bit〜16bit寄りのシンセ／チップチューン系で、暗めのファンタジー世界観。」

## 2. 画面・UIのテイスト

| 項目 | 内容 |

## 3. ステージ別テーマ（背景・タイル・雰囲気）

各ステージは「theme」で背景色・タイル種類が切り替わります。画像・BGMをステージ単位で依頼するときの参照用です。

| Stage | テーマ名 | 雰囲気 | 背景グラデーション目安 | タイル・オブジェクト |
|-------|-----------|--------|------------------------|----------------------|
| **1** | grass | 冒険の始まり、草原・土の道 | #0a0a2e → #1a1a4e → #2a2a5e（青紫） | 草原(grassTop)、土(dirt)、スパイク、土台 |
| **2** | grass | 同じ草原世界、やや先へ | 上に同じ | 同上＋コイン・ポーション・動く足場 |
| **3** | ice | 氷のステージ、静謐 | #0a1a3e → #1a2a5e → #2a3a6e（寒色） | 氷(iceTop, iceBlock)、青白いオーラ |
| **4** | metal | 機械・メカニカル | #0a0a15 → #151525 → #202035（暗いグレー） | 金属(metalTop, metalBlock)、機械的な足場 |
| **5** | castle | 城・ダークファンタジー | #0a0810 → #1a1520 → #252030（紫黒） | 城壁(castleTop, castleBlock) |
| **6** | dark | 闇・最終ボス前 | #0a0008 → #150010 → #200818（赤黒） | 闇(darkTop, darkBlock)、溶岩(lavaTop)も使用。背景に赤い光点の演出あり |

**共通の背景要素**  
- 星（白、小さく、ゆらぎあり）  
- 遠景の山（シルエット、themeで色変更）  
- 遠景の木（シルエット、themeで色・形変更：metalは四角め、darkは不気味な形）

**指示文の例（背景用）**  
「Stage 3は氷ステージ。背景は寒色の青グラデ（#0a1a3e〜#2a3a6e）。星とシルエットの山・木。16pxベースのピクセルアート、横スクロール用パララックス用。」

## 6. 音楽・効果音

### 6.1 効果音（内蔵ウェブ音声・se.js）

Web Audio API で合成した効果音（外部ファイル不要）を以下のタイミングで再生しています。

| 効果音 | タイミング |
|--------|------------|
| ジャンプ | 主人公がジャンプした瞬間 |
| 剣（スラッシュ） | 通常攻撃で剣を振った瞬間 |
| 剣ビーム | 溜め撃ちで波動を放った瞬間 |
| コイン | コインを取得したとき |
| ポーション | 回復アイテムを取ったとき |
| パワーアップ | パワーアップを取ったとき |
| 主人公被弾 | ダメージを受けたとき |
| 雑魚撃破 | 敵を倒したとき |
| ボス撃破 | ボスを倒したとき（Phase2移行時は鳴らさない） |
| ゲームオーバー | 体力0になったとき |
| ステージクリア | ゴールの旗に触れたとき |

### 6.2 BGM 指示のベース

現在のステージとBGMの対応は以下の想定です。

| ステージ | 曲の役割 | 雰囲気のキーワード |
|----------|----------|---------------------|
| 1, 2 | 穏やか・冒険の始まり | 朝〜昼の草原、希望、少しワクワク、まだ戦闘は軽め |
| 3 | 静謐（氷） | 冷たい、静か、神秘、魔法の森・氷の洞窟 |
| 4, 5 | 決意・戦闘 | 緊張、メカニカル or 城の重厚感、ボス前の覚悟 |
| 6 | クライマックス・ボス | 最終戦、暗く重い、魔王、怒涛の展開 |

**指示文の例（BGM用）**  
「2D横スクロールアクション用BGM。Stage 1–2は穏やかで冒険感、Stage 3は静かで冷たい氷ステージ、Stage 4–5は戦闘・決意、Stage 6は最終ボス用のクライマックス。8bit〜16bit風、ループ可能、1〜2分程度。」

## 6.5 Suno 用 BGM プロンプト（ボス戦・プロローグ・エンディング）

ゲームの世界観に合わせて Suno に渡すプロンプト例です。曲の役割ごとに「Style of music」や「Lyrics」欄に貼り付けて使えます。**インスト曲**を想定しているため、歌詞は空欄または "instrumental" でよいです。

---

### ボス戦用 BGM（Stage 6 最終ボス・その他ボス戦にも流用可）

**Suno に入力する説明文（英語推奨）**

```
16-bit retro action game boss battle theme. Dark fantasy, sword and magic. Intense and dramatic. Night atmosphere, red and black tones. Builds tension, powerful drums and bass, epic synth leads. Not comical, slightly serious. Loopable 1-2 minutes. Chiptune, 8-bit 16-bit style. No vocals, instrumental.
```

**日本語メモ（依頼時の補足用）**  
最終ステージ・魔王戦のクライマックス。暗く重い、怒涛の展開。赤黒を基調にした闘いのオーラ。ループ可能で1〜2分。

---

### プロローグ用 BGM（タイトル〜冒険の始まり）

**Suno に入力する説明文（英語推奨）**

```
16-bit retro RPG prologue and title screen music. Calm, hopeful, a little mysterious. Dawn or twilight adventure begins. Pixel art fantasy world, sword and magic. Gentle melody, warm pads, not too bright. Slightly serious, nostalgic. Loopable 1-2 minutes. Chiptune, 8-bit 16-bit style. No vocals, instrumental.
```

**日本語メモ（依頼時の補足用）**  
タイトル画面・プロローグ用。穏やかで希望を感じる、少し神秘的な「冒険の始まり」。夜〜夕暮れの雰囲気。過度に明るくせず、懐かしさのあるレトロファンタジー。

---

### エンディング用 BGM（オールクリア後）

**Suno に入力する説明文（英語推奨）**

```
16-bit retro game ending theme. Victory, journey complete. Warm and triumphant but not cheesy. Dark fantasy adventure concluded. Golden hour feeling, relief and pride. Gentle build, satisfying resolution. Loopable or 1-2 min one-shot. Chiptune, 8-bit 16-bit style. No vocals, instrumental.
```

**日本語メモ（依頼時の補足用）**  
全ステージクリア後のエンディング。勝利と旅の終わり。温かく達成感があるが、やりすぎない程度に。ゴールドの光のような安堵と誇り。ループまたは1〜2分の一曲完結でOK。

## 7. 依頼時のチェックリスト（画像・音楽）

- [ ] **色・キーワード**：上記表から該当する色コードやキーワードをコピーして追記
